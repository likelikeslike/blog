学习日语形容词

形容词分类

い形容词

修饰名词:
～い + 名词

作谓语:
现在:

  • 肯定: ～い + です
  • 否定: ～い　→ 〜く +　ないです

过去:

  • 肯定: 去掉い + かったです
  • 否定: ～い　→ 〜く + なかったです

好: 良いい 和 良よい 都从 良よい 变化
良よい → 良よくないです 不好 (现在)
良よい → 良よかったです 好 (过去)
良よい → 良よくなかったです 不好 (过去)

な形容词

修饰名词:
词干 + な + 名词

作谓语:
现在:

  • 肯定: 词干 (去掉な) + です
  • 否定: 词干 (去掉な) + では ありません

过去:

  • 肯定: 词干 (去掉な) + でした
  • 否定: 词干 (去掉な) + では ありませんでした
