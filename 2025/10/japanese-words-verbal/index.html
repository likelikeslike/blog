学习日语动词

动词分类

动词的去ます形: 动词ます形去掉ます, 剩下的不变
动词的て形: 链接两个或两个以上的动作

一类动词

う段假名结尾的动词: う、く、ぐ、す、つ、ぬ、ぶ、む

ます形: 结尾的う段假名 → い段假名 + ます

て形:

  • う、つ、る 结尾 → 去掉 う、つ、る + って
  • く、ぐ 结尾 → 去掉 く、ぐ + いて / いで
  • ね、ぶ、む 结尾 → 去掉 ね、ぶ、む + んで
  • す 结尾 → 去掉 す + して

行いく 变て形: 行いって

二类动词

る结尾的动词 (い段/え段 + る结尾, 90%为二类动词)

ます形: 去掉 る + ます

て形:　去掉 る + て

三类动词

只有两个:

  • する
  • 来くる

ます形:
する → します
来くる → 来きます

て形:
する → して
来くる → 来きて

双汉字名词/片假名名词 + する 也是三类动词, 变形方式与 する 一致
