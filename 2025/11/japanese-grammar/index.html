学习日语语法

句型

判断

句型 含义
X は Y です X 是 Y
X は Y ではありません X 不是 Y
あまり V(否定)/A(否定) 不太…

存在

句型 含义
N が あります 有xx (无生命)
N が います 有xx (有生命)
N は N(地点) に あります xx (无生命) 在某处
N は N(地点) に います xx (有生命) 在某处
N(地点) に N　が あります 某处有xx (无生命)
N(地点) に N　が います 某处有xx (有生命)

动作

句型 含义 备注
N を V 做…
N(地点) で V 在哪里做…
N(时间) (に) V 什么时候做… 确切时间点时加 に
N(主) は N(人物) に N(物) を あげます xx给谁…
N(主) は N(人物) から N(物) を 貰もらいます xx从谁得到…
N(主) は 私 に N(物) を くれなす xx给我… 被动接受
N(主) は N(人物) に N(物) を 借かります xx从谁借到…
N(主) は N(人物) に N(物) を 貸かします xx给谁借出…
N(工具/手段) で V 通过工具/手段做…
N に 会あいます 和…见面 に 提示对象
N(地点) を V を 表示动作出发点或经过某个地方
N(地点) に V に 表示动作终止点
V(原形) 前まえに 做…之前

请求/建议

句型 含义
N を ください 请给我…
V ませんか 做…好吗
V ましょう 做…吧
V ましょうか 为你做…好吗
A(～い/～な) ほうが いいです 最好怎么样

目的

句型 含义
N(地点) へ 行きます / 来ます / 帰ります 去/来/回…
N(地点) へ V(去ます形) に 行きます / 来ます / 帰ります 去/来/回xx做…
V(去ます形) に 行きます 去做…

动词て形

此处 V 均为动词て形 て 之前的部分, 下方 て 需要根据变形规则改为 で

句型 含义 备注
V て から V 做完xx之后做xx 只能是两个动作
V て ください 请做…
V ても いいです 可以做…
V ては いけません 不可以做…
V て います 正在做…
V て V 相继发生动词并列 可以多个动作

动词ない形

此处 V 均为动词ない形 ない 之前的部分

句型 含义 备注
V ない で ください 请不要做…
V なければ なりません 必须做…
V なくても いいです 不做…也可以
V ない ほうが いいです 最好不做…

动词た形

此处 V 均为动词ない形 た 之前的部分, 下方 た 需要根据变形规则改为 だ

句型 含义 备注
V た ことが あります 做过…
V た 後あとで 做完…之后
V た ほうが いいです 最好做…

喜好/能力/欲望

表示喜好, 能力, 欲望, 存在, 情感的形容词或动词做谓语时提示宾语的助词用 が

句型 含义
N が 好きです 喜欢…
N が 欲しいです 想要…
N が 嫌います 讨厌…
N が 怖こわい …很吓人
N を V(去ます形) たいです 想做…
N が 分かります 懂…
N が できます 会…
V(原形) こと が できます 能做…

原因

句型 含义
句子 から。 因为…
句子1。だから 句子2。 だから 之前的句子表示原因, だから 后面表示结果

列举

句型 含义 备注
N と N xx和xx 完全列举
N や N や N など xx和xx和xx等等 不完全列举, 最后 など 表示等等
N か N xx或xx 二者选一

比较句型

句型 含义 备注
N は N より A です xx比xx怎么样 此处形容词为 ～い 或 な形容词词干
N より N のほうが A です xx比xx更怎么样 此处形容词为 ～い 或 な形容词词干
N と N と どちらが A ですか xx和xx哪个更怎样 此处形容词为 ～い 或 な形容词词干
N の中なかで N/疑问词 が 一番 A です xx之中 xx/谁/什么 是最怎么样的 中で 的で表示范围
疑问词 が 一番 A ですか xx是最怎么样的
N は N ほど A(否定) xx不如xx怎么样 ほど 之前的名词是参照物

数量词/频率

句型 含义 备注
N を 数量词 V 做几个xx 数量词加在动词之前
N(时间段) に 次数 V xx时间内做什么几次
数量词 で 以xx个数量为一组 一个的时候不加 で

其他

句型 含义 备注
句子1 が、句子2 句子1, 但是… が 表示转折
まだ 否定句 还没…
疑问词 でも 全都…
疑问词 か 不确定的提问
N に よって 根据…/由于… 也可用 N に よります 放在句尾
N　について 关于…

简体形和敬体形

敬体形: 〜 ます, 〜 です, 用于比自己身份地位高的人
简体形: 普通形, 用于口语及比自己辈分低的人, 包括动词, 形容词, 名词及其现在/过去的肯定/否定形式

现在肯定 现在否定 过去肯定 过去否定
动词 原形 ない 形 た 形 なかった 形
い形容词 原形(～ い) 〜 くない 〜 かった 〜 くなっかた
な形容词/名词 词干/名词 + だ 词干/名词 + ではない 词干/名词 + だった 词干/名词 + ではなっかた

表示并列的句型

た形(普通型过去肯定) + り + た形(普通型过去肯定) + り + します (动词) / です (形容词/名词)

  • 动词时表示: 又做…又做…; 有时做…, 有时做…
  • 形容词/名词时表示: 有的…, 有的…

使用形容词时, 两个形容词要是反义词

表示不确定的句型

动词普通型/い形容词/な形容词词干/名词 + か どうか

表示对所说的不确定, 后常接 分かりません / 知りません

表示意愿的句型:

我(其他人)认为/想…: 普通形 + と 思います / 思いました / 思っています

  • 思います 只能用于第一人称
  • 思いました / 思っています 即可用于第一人称也可用于第三人称

转述他人话语:

  • 间接转述: 名词(人) は + 普通型 + と 言いますた
  • 直接转述: 名词(人) は + 普通型 + と 言っています

表示强调原因或理由:

  • 口语: 普通型 + んです
  • 书面语: 普通型 + のです

当前面是名词/な形容词时, だ → な
